GET /api/interests : liste des intérêts

GET /api/sessions : liste des sessions

GET /api/session/{id} : détail de la session portant l'id défini

GET /api/session/{id}/comments : commentaires de la session portant l'id défini

POST /api/session/{id}/comment : ajoute un commentaire à la session portant l'id défini (Connecté)

GET /api/lightnings : liste des lightning talks

GET /api/lightning/{id} : détail du lightning talk portant l'id défini

GET /api/session/{id}/comments : commentaires du lightning talk portant l'id défini

POST /api/lightning/{id}/comment : ajoute un commentaire au lightning talk portant l'id défini (Connecté)

GET /api/members : liste des membres (Connecté)

GET /api/member/{id} : détail d'un membre (Connecté)
